# Mislo

Mislo - simply interpritator for mislo language.

## Commands
```
List of commands:
	write var_name - write text to console.
	accept var_name - get value from user and write in var.
	help - show this message.
	exit - leave from program.
	clear - clear console output.
	quit - alias for exit.
```

## Quick start
```
# clone repo
git clone https://github.com/mental0-main/mislo.git

# move to dir
cd mislo

# Build
cmake . && make -j4

# Run
./mislo

```

## Requirements
- Clang/Gcc
- Git
- Cmake
- Make
